
# Okeanos Sample Config

## Snap all windows to grid

Cmd + Shift + m

## Move Window

Cmd + Shift + hjkl

## Resize Window

Cmd + Ctrl + hjkl

## Change Grid Size

**Horizontal**
- Cmd + Ctrl + Shift + Alt + z
- Cmd + Ctrl + Shift + Alt + x

**Verticle**

- Cmd + Ctrl + Shift + Alt + a
- Cmd + Ctrl + Shift + Alt + r

## Change Padding

Cmd + Ctrl + Shift + Alt + -
Cmd + Ctrl + Shift + Alt + =

## Maximize

Cmd + Alt + n

## Minimize to Dock

Cmd + Alt + o

## Switch Screen

Cmd + Shift + e
